Installation

Download the most recent official release at Github. Don’t unzip the downloaded zip.

To install the add-on, navigate to Edit > Preferences > Add-Ons. Then seach for Rigify Gamerig Extension using the search bar. Finally activate the add-on by pressing the checkbox.

Note

Ensure to also have Rigify enabled, otherwise the add-on will never show up.
