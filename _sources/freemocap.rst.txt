Freemocap
=========

The Free Motion Capture Project `Freemocap
<https://freemocap.org>`_
aims to provide research-grade
markerless motion capture software to everyone for free. 

It features multicam detection using Mediapipe and is currently in alpha.
You can import generated **session data** using the freemocap data import options.


Path
    Select the directory path to the *freemocap session folder directory*.

Load Session folder
    Loads the tracking results.

Load synchronized videos
    Load session videos as planes.

Quickload
    Disable Viewport update while importing.
    While this improves the import speed drastically, it will freeze blender.

    Raw
        Import the raw data which may not be used to transfer.
        Mainly for scientific purposes.
